# 🚀 Release Notes - Orbit v1.0.0 (Official Launch)

We are thrilled to present the initial official launch of **Orbit**! 🌌

Orbit is a premium, system-level installer and orchestrator desktop app for mobile developers, agent configurations, and **Antigravity IDE** integrations.

---

## 🔥 What's New in v1.0.0

### 🚀 Bootstrapping Stacks
- **Express Installation Mode**: Automatically installs and configures Node.js, Git, Python 3, Supabase, EAS CLI, and OrbitPrompter CLI in a single click.
- **Custom Payload Mode**: Enables developer choice of which clis, frameworks, and workspace configurations to deploy.

### 🛡️ Real-time Diagnostics
- Integrated local machine validator checking global paths, CLI presence, and library paths.
- Clean health status checkmarks displaying exact components in a unified, premium card list.

### 🤖 Autopilot Scaffolding
- Instant workspace scaffold deployments under user-specified directories (`~/.remoat/config.json`).
- Automatically setups `~/.agents/workflows/` with pre-defined robust workflow commands (`feature.md`, `submit.md`, `test.md`, `project-creator.md`).

### 🎛️ Bot Orchestrator
- Control panel to launch, terminate, and query the **OrbitPrompter** bot (`remoat`) directly from Orbit.

### 🎨 Visual & Aesthetic Overhaul
- **Brand New Icon**: Refreshed the entire UI to use the premium, customized `orbit_icon.svg` as the default favicon, app brand, and window icon.
- **Micro-animations**: Dynamic CSS glassmorphic cards, active loading glow-state transitions, and spring animations on interaction.

## 🛠️ Installation Instructions

### macOS (Darwin)
1. Download the `.dmg` package from the Assets list.
2. Mount the `.dmg` volume.
3. Drag `Orbit` into your local `/Applications` folder.
4. Launch `Orbit` (if macOS shows developer verification warnings, right-click -> Open).

### Windows (Win32)
1. Download the `.exe` installer package from the Assets list.
2. Execute the installer and proceed with the wizard prompts.
3. Launch `Orbit` from your Desktop or Start Menu.

---

## 📝 Requirements
- **macOS** 10.15+ / **Windows** 10+
- Node.js (>= 18.x) and git installed globally on host path.
